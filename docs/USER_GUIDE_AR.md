# دليل الاستخدام

دليل عملي موجز لاستخدام المنصّة خلال البرنامج الصيفي.

## الدخول
- افتح رابط المنصّة وسجّل الدخول باسم المستخدم وكلمة المرور.
- من «الملف الشخصي» يمكنك تغيير كلمة المرور وتسجيل الخروج.

## اختيار الأسبوع
- في الأعلى يوجد محدّد الأسبوع. كل أسبوع **مستقل تمامًا** عن غيره.
- التقييم والسحب والتميّز والتقارير كلّها تخصّ الأسبوع المختار.

## اللوحة
- نظرة سريعة: عدد الطلاب، الحلقات المكتملة/المعلّقة، المؤهلون، المستبعدون، نسبة التقدّم.
- «نشاط مباشر» يعرض آخر الإجراءات لحظيًا.

## الحلقات
- قائمة الحلقات مع حالة كل حلقة ونسبة إكمال تقييمها.
- اضغط حلقة لعرض طلابها وحالتهم، وإضافة/نقل/حذف طالب (حسب صلاحيتك).
- الطالب المستبعد يظهر بلون رمادي مع سبب الاستبعاد.

## التقييم (الأهم)
1. اختر الحلقة ثم المعيار من التبويبات: الحضور، المظهر، السلوك، المنهج.
2. للمعايير اليومية (الحضور/المظهر/السلوك) تظهر شبكة أيام **الأحد–الخميس**.
   للمنهج تظهر حالة أسبوعية واحدة.
3. انقر الخلية للتبديل:
   - نقطة «·» = غير مُدخل
   - «✓» = سليم
   - «✗» = مخالفة
4. **الحفظ تلقائي فوري** — لا يوجد زرّ حفظ. مؤشّر الحفظ يظهر أعلى الصفحة.
5. تتحدّث **أهلية الطالب لحظيًا** في عمود «الأهلية»:
   - أي مخالفة في الحضور/المظهر/السلوك ← **استبعاد من السحب والتميّز**.
   - مخالفة المنهج فقط ← **استبعاد من التميّز** مع بقاء الأهلية للسحب.

> إذا كان زميل آخر يحرّر نفس الحلقة، يظهر تنبيه بذلك أعلى الشبكة.

## التميّز الأسبوعي
- لكل حلقة **متميّز واحد** في الأسبوع.
- إن كان هناك **مؤهل واحد فقط** يُسجّل تلقائيًا.
- إن تعدّد المؤهلون: تظهر «بانتظار الحسم»، اضغط «اختيار المتميّز» (بعد استشارة المعلّم
  خارج النظام) واختر طالبًا.
- إن لم يوجد مؤهل: تظهر «لا مؤهل».

## السحب الأسبوعي
- لكل **مجموعة** (5 مجموعات) سحب مستقل.
- اضبط عدد الفائزين (الافتراضي 3 ويُعدّل من الإعدادات) ثم «ابدأ السحب».
- النتيجة تظهر كمسودّة ويمكن «إعادة السحب». عند الاطمئنان اضغط **«اعتماد»**
  (الاعتماد نهائي ولا يمكن تعديله).
- «فتح شاشة العرض» يفتح شاشة بروجكتر تعرض الكشف المتحرّك للفائزين لحظة السحب.

## التقارير
- ملخّص الأسبوع: الفائزون بالسحب لكل مجموعة، متميّز كل حلقة، المستبعدون وأسبابهم.
- «تصدير PDF» يفتح نسخة مهيّأة للطباعة — اطبعها أو احفظها PDF من المتصفّح.

## المستخدمون (للمدير)
- أنشئ حسابات المساعدين وحدّد لكلٍّ:
  - **الحلقات المُسندة** (نطاق عمله)، و**الصلاحيات** (مثل تقييم الحضور فقط، أو السحب).
- يمكن تعطيل حساب فيُنهى دخوله فورًا.

## الإعدادات (للمدير)
- العدد الافتراضي للفائزين بالسحب.
- استيراد الطلاب من ملف Excel (عمودان: الاسم، الحلقة).
- معلومات النسخ الاحتياطي.

## سجل النشاط
- سجل تدقيق كامل لكل الإجراءات مع إمكانية التصفية حسب نوع الإجراء.

## أسئلة شائعة
- **هل أحتاج للحفظ؟** لا، كل تغيير يُحفظ تلقائيًا.
- **استبعدت طالبًا بالخطأ؟** أعد النقر على الخلية حتى تعود لـ«✓ سليم» أو «·».
- **هل تتأثر بقية الأسابيع؟** لا، كل أسبوع مستقل.
