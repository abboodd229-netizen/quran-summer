# دليل التثبيت

## 1) المتطلبات
- **Node.js 20 أو أحدث** و **npm 10+**.
- نظام يدعم بناء وحدات أصلية (better-sqlite3 يُبنى عبر node-gyp تلقائيًا عند التثبيت).
  على لينكس: عادةً جاهز. على ويندوز: ثبّت "Desktop development with C++" أو استخدم WSL.

تحقّق:
```bash
node -v   # v20+
npm -v    # 10+
```

## 2) تثبيت الحزم
من جذر المشروع:
```bash
npm install
```
هذا يثبّت حزم workspaces الثلاث (shared/server/client) ويبني الحزمة المشتركة تلقائيًا
(عبر postinstall). إن ظهر خطأ في بناء better-sqlite3، ثبّت أدوات البناء ثم أعد المحاولة.

## 3) إعداد متغيّرات البيئة
انسخ المثال وعدّل القيم:
```bash
cp .env.example server/.env
```
أهم المتغيّرات:
- `SESSION_SECRET` — **غيّره لقيمة عشوائية طويلة** في الإنتاج.
- `ADMIN_USERNAME` / `ADMIN_PASSWORD` — حساب المدير الأولي.
- `DATABASE_PATH` — مسار ملف SQLite (افتراضي `./data/app.sqlite`).
- `PORT` — منفذ الخادم (افتراضي 8080).

> ملاحظة: عند التشغيل عبر `npm ... -w server` يكون مجلد العمل هو `server/`،
> لذا `./data/app.sqlite` يقع داخل `server/data/`.

## 4) تهيئة قاعدة البيانات (بذور)
```bash
npm run seed
```
يُنشئ: 5 مجموعات، 14 حلقة، 4 أسابيع، 204 طالب (من `server/src/db/seed/data.json`)،
وحساب المدير. آمن للتكرار: لن يُكرّر الطلاب إن كانوا موجودين.

> الخادم أيضًا يهيّئ المجموعات/الحلقات/الأسابيع تلقائيًا عند الإقلاع، لكن الطلاب
> تُضاف فقط عبر `npm run seed` أو الاستيراد من Excel داخل التطبيق.

## 5) التشغيل
### تطوير (إعادة تحميل تلقائي)
```bash
npm run dev
```
- الخادم: `http://localhost:8080`
- الواجهة: `http://localhost:5173` (تُمرّر طلبات `/api` للخادم تلقائيًا)

### إنتاج محلي
```bash
npm run build
npm start
```
ثم افتح `http://localhost:8080` (الخادم يقدّم الواجهة المبنية).

## 6) أول دخول
سجّل الدخول بحساب المدير من `.env`، ثم:
1. غيّر كلمة المرور من «الملف الشخصي».
2. أنشئ حسابات المساعدين وأسند لهم الحلقات والصلاحيات.
3. ابدأ إدخال التقييم.

## استكشاف الأخطاء
- **فشل بناء better-sqlite3**: ثبّت أدوات بناء C++، أو استخدم Node 20 LTS، ثم `npm rebuild better-sqlite3`.
- **الواجهة لا تتصل بالـ API في التطوير**: تأكّد أن الخادم يعمل على 8080 (إعداد proxy في `client/vite.config.ts`).
- **القاعدة فارغة من الطلاب**: شغّل `npm run seed`.
