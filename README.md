# منصّة التحفيز والمكافآت الأسبوعية — الحلقات الصيفية 1448هـ

**الإصدار 1.0** | الجمعية الخيرية لتحفيظ القرآن الكريم بجازان

تطبيق تشغيلي لأتمتة التحفيز والمكافآت الأسبوعية في البرنامج الصيفي لتحفيظ القرآن.
البرنامج: شهر واحد، أربعة أسابيع مستقلّة، 204 طالب موزّعون على 14 حلقة و5 مجموعات سحب.

> ليس نظام تعلّم/إدارة حفظ — بل أداة عمليات: تقييم أسبوعي، أهلية تلقائية، سحب
> عشوائي عادل، متميّز لكل حلقة، تقارير وطباعة.

---

## أبرز القدرات

- **محرّك الأهلية** (لكل أسبوع، مستقل): مؤهل للسحب = لا مخالفة في الحضور/المظهر/السلوك.
  مؤهل للتميّز = مؤهل للسحب + لا مخالفة منهج.
- **التقييم**: شبكة أيام (الأحد–الخميس) للحضور/المظهر/السلوك، ومفتاح أسبوعي للمنهج،
  مع **حفظ تلقائي فوري** وإعادة حساب الأهلية لحظيًا.
- **السحب**: لكل مجموعة (5 مجموعات)، عشوائية تشفيرية بلا تكرار، مسودّة ← اعتماد نهائي،
  وشاشة عرض للبروجكتر بكشف متحرّك عبر SSE.
- **التميّز**: فائز واحد لكل حلقة أسبوعيًا (تلقائي عند مؤهل واحد، يدوي عند التعدّد).
- **الأدوار والصلاحيات**: مدير + مساعدون بنطاق حلقات وصلاحيات دقيقة.
- **تقارير**: ملخّص أسبوعي + تصدير PDF (طباعة من المتصفّح بقالب عربي RTL).
- **سجل تدقيق** لكل إجراء، **أقفال تحرير ناعمة** لتفادي التعارض، **نسخ احتياطي يومي**.

## التقنيات

| الطبقة | التقنية |
|---|---|
| الواجهة | React 18 + TypeScript + Vite + Tailwind (RTL) + React Query + React Router |
| الخادم | Node 20 + Express + TypeScript |
| البيانات | SQLite عبر better-sqlite3 (WAL) |
| المصادقة | جلسات في قاعدة البيانات + كوكي httpOnly + bcrypt |
| اللحظي | Server-Sent Events (SSE) |
| التحقق | Zod (مشترك بين الطرفين) |

## بنية المشروع

```
quran-summer/
├─ shared/        حزمة مشتركة: الثوابت، مخططات Zod، الأنواع
├─ server/        الخادم (Express + SQLite)
│  └─ src/
│     ├─ config/  البيئة والسجل
│     ├─ db/      المخطط، الاتصال، البذور (seed/data.json = 204 طالب حقيقي)
│     ├─ core/    المحرّكات: الأهلية، السحب، الحالة، التدقيق، SSE، النسخ
│     ├─ middlewares/  المصادقة، التحقق، الأخطاء
│     └─ modules/ auth, users, catalog, students, events, lottery,
│                 excellence, reports, dashboard, audit, locks, settings, stream
└─ client/        الواجهة (Vite + React)
   └─ src/
      ├─ app/      السياقات، الهيكل، التوجيه
      ├─ components/ مكوّنات واجهة قابلة لإعادة الاستخدام
      └─ features/  الشاشات (لوحة، حلقات، تقييم، تميّز، سحب، تقارير، ...)
```

## تشغيل سريع (محليًا)

المتطلبات: **Node.js 20+**.

```bash
npm install            # يثبّت كل الحزم ويبني الحزمة المشتركة
npm run seed           # يهيّئ القاعدة: مجموعات/حلقات/أسابيع + 204 طالب + مدير
npm run dev            # يشغّل الخادم (8080) والواجهة (5173) معًا
```

ثم افتح `http://localhost:5173`. حساب المدير الأولي:
`admin` / `Admin@12345` — **غيّر كلمة المرور بعد أول دخول**.

للإنتاج محليًا:

```bash
npm run build          # يبني shared ثم client ثم server
npm start              # يشغّل الخادم ويقدّم الواجهة المبنية على المنفذ 8080
```
